APIdays 2013

The web 1.0 was readable, the web 2.0 was social, now the web is programmable trough application programming interfaces, aka apis.

“Software is eating the world” said Marc Andreessen in the early 2000.

Since few years, web Applications Programming Interfaces, aka APIs are disrupting this Software industry by defining either a new technical and commercial access and relation between applications developers and companies.

These technical interfaces completely change the partnership and developer relationship for deep integration in a self-service model that leverage the accessibility and the openness of the web.

APIs can reduce global integration process time from months to minutes and can leverage an ecosystem of applications in an effective lean open innovation model.

These technical and business interfaces made the success of the new giants of the IT industry.

APIdays annual conference, with the presence or worldwide speakers will present all the opportunities to better understand paradigms behind APIs, thinking the future of Software through APIs, on Cloud, Big Data and Internet of Things topics.

Now we can say, as Steven Willmott CEO of 3Scale, during last APIdays conference in San Francisco that : "if Software is eating the world, APIs are eating software.”
